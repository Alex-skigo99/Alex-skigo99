Full Stack Application · B2B · AI-Powered

Personality Collection

A full-stack web platform that assesses YouTube creators' personality types through a structured Big 5 personality survey and delivers AI-generated, personalised growth recommendations. Designed as a B2B tool for agencies or networks that manage YouTuber talent.

View Source
React 18 React Router 6 Redux Toolkit TanStack React Query Axios Tailwind CSS Ant Design Node.js Express MongoDB Mongoose Groq Cloud (Llama 3.3 70B) AWS SES Jest
Business Logic
  • Invitation Flow — YouTuber receives a branded email with a unique survey link containing their email and channel URL
  • Survey Instance Creation — When the email logo loads (image tracking pixel), the system creates a new survey record
  • Personality Assessment — Big 5 Personality Model survey with individual answer saving for resume support
  • Engagement Mechanics — "Meme breaks" every 5 questions with progress-based encouragement
  • Channel Verification — Post-survey modal for YouTube channel ownership confirmation
  • AI-Powered Results — LLM generates: creator type, superpower, weakness, growth tip, summary, and strategy
  • Viral Sharing — Share survey with other creators via email from the result page
Architecture

Key Design Decisions

  • Individual Answer Saving — Each response PATCHed independently for data safety and resume support
  • React Query over Redux for Server State — Survey data via React Query; Redux minimal (loader only)
  • Interceptor Pattern — Single InterceptorsComponent with Axios interceptors and global Spin overlay
  • Error Class Hierarchy — Base HRCustomError with domain-specific subclasses
  • Cached AI Results — Report generated once, stored in survey document, cached on subsequent requests
  • Email Tracking Pixel — Logo URL contains creator info; image load triggers survey creation
  • In-Memory MongoDB for Tests — mongodb-memory-server with disposable replica set
